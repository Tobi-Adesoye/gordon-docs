Gordon Engine API

High-speed matrix optimization service for production-grade workloads.

Endpoint

POST https://gordon-core-engine.onrender.com/api/v1/optimize

Request Format

Send your data as a JSON object with the following schema:

{
  "shape": [3],
  "matrix_data": [1.0, 2.0, 3.0]
}

Authentication

Include your API key in the header:

Authorization: Bearer YOUR_SECRET_KEY
